# Task-4
This project focuses on data cleaning and improving data quality to prepare a dataset for accurate analysis. The dataset was examined to identify issues such as missing values, duplicate records, inconsistent formats, and potential outliers.

Various data preprocessing techniques were applied, including handling missing data, removing duplicate entries, standardizing column formats, and detecting outliers using statistical methods. These steps ensured the accuracy, consistency, and reliability of the dataset.

The cleaned dataset becomes more suitable for further analysis, reporting, and machine learning tasks, helping generate more reliable insights and better decision-making.
